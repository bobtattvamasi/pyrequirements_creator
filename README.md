# pyrequirements_creator

![](https://packagecontrol.io/readmes/img/34e2cf8c12621c907b0b7dbea924aec64ec3c4e3.gif)

Использование скрипта:
```python
python requirements_creator.py <your_folder_name>
```
Т.е. вызвать скрипт через питон, указав путь к проекту.
В итоге в корне проекта будет создан файл requirements.txt
